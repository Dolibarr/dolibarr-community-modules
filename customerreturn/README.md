# Customerreturn - Customer Returns

Customer merchandise return management with shipment-initiated returns.

- Per-line quantity tracking from original shipments
- Stock movements on receive (automatic re-entry to warehouse)
- Credit note generation from returns
- 3-status lifecycle: Draft, Validated, Received
- Integrates with WarrantySvc for RMA-initiated returns

**Repository:** https://github.com/zacharymelo/doli-returns
