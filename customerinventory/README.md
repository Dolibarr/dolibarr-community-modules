# Customerinventory - Customer Inventory

Adds a Customer Inventory tab to third-party cards showing all products shipped to a customer.

- Flat list or grouped views (by sales order, invoice, or product)
- Serial/lot number display per shipment line
- Multiple invoice links per row when orders span invoices
- Badge count on tab showing total inventory items
- Optional integration with Customer Returns module for return status

**Repository:** https://github.com/zacharymelo/dolibarr-customer-inventory
