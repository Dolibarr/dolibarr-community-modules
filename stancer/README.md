# STANCER FOR [DOLIBARR ERP & CRM](https://www.dolibarr.org)

## Features

Integrates the Stancer payment mode inside Dolibarr.

Allow to use a Stancer account to pay any membership dues, donation, customer invoice or customer order with the PayPlug API.

Please follow that link to open your account on Stancer : https://stancer.com/fr/paiement-en-ligne?mtm_campaign=Dolibarr&mtm_source=Dolibarr
