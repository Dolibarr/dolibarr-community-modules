# WarrantySvc - Warranty & Service Management

RMA (Return Merchandise Authorization) and Warranty Management for serialized equipment.

- Track warranties by serial number with configurable warranty types and coverage durations
- Manage service requests with full lifecycle tracking
- Auto-create warranties on shipment validation or order close
- PDF authorization slips for service requests
- Optional integration with Customer Returns module
